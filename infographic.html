1

Transfer Learning Setup

The ResNet50 base model was loaded, **pre-trained on ImageNet**. This provides a strong foundation of image features, saving huge training time.

  • Data Augmentation Applied (Rotation, Zoom)
  • Weighted Loss to address class imbalance
2

Phase 1: Freeze & Train

The ResNet50 base was **frozen** to lock in pre-trained weights. Only the newly added classification **Dense Head** was trained using a standard learning rate.

  • Goal: Quickly adapt the output layer.
  • Prevented catastrophic forgetting.
3

Phase 2: Deep Fine-Tuning

The entire ResNet50 base was **unfrozen**. Training continued with an extremely **low learning rate** ($1e-5$).

  • Goal: Tweak low-level X-ray features.
  • Crucial step to avoid the "Class Collapse" bug.
4

Threshold & Deployment

The model's probability outputs were tuned. We found the **Optimal Decision Threshold** of **$0.700$** to maximize balanced accuracy.

  • Model saved as: `ResNet50_Medical_XRay_final.h5`
  • Threshold saved to: `decision_threshold.txt`

Project successful and model deployed. Ready to classify new images using the saved configuration.
